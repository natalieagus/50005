### First Week Checklist

If you are just starting the course, make sure you do the following before your first lab session:

1. Set up a POSIX-compliant OS (see [Getting Started]({{ site.baseurl }}/admin/getting-started))
2. Join the ED discussion forum (link on eDimension)
3. Form your PA and lab checkoff team (2 to 3 pax, same cohort) via link on eDimension by the end of Week 1
4. Read the [Assessment]({{ site.baseurl }}/admin/assessment) page so you know what counts and how much
